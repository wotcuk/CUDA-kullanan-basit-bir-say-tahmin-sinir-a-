# CUDA-kullanan-basit-bir-say-tahmin-sinir-a-
CUDA kullanarak basit bir nöral ağ yazdım çok birşey beklemeyin.
28x28 lik beyaz yüzey üzerine siyah 0dan 9a kadar çizilen rakamları tahmin ediyor. 
Başka boyutta kullanılan resimler içinse otomatik uygun boyutlarda nöral ağ oluşturup eğitime başlıyor.
Eğitim verisi için görsellerin isimleri Tur(3)Sayi(2).bmp gibi olmalı tur sayısını kendiniz derlerken değiştirebilirsiniz sayı içersine yazılan değerse görseldeki sayı olarak eğitiliyor.
